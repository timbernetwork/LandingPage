Timber Network

Timber Network enables cross-chain interoperability by securely delivering messages between blockchains.

Timber Protocol Whitepaper v3 Timber Voting Whitepaper 0.1 Github


Timber Network is a blockchain interoperable messaging system which enables blockchains to securely interact with one another. Timber is powered by a network of relayer nodes and is secured by Tendermint. Anyone is free to use Timber Network.

Secret Ballot DAO Voting on Ethereum

In this example Timber leverages the privacy of Secret Network to give DAOs on Ethereum secret ballot voting, without ever even having to bridge their ERC20 voting tokens. Blockchains shouldn't just talk, they should harmonize. More Info.
