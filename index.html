Timber Network

Trustless Incentivized Messaging Blockchain Encrypted Relayer.



Github


Timber Network is a blockchain interoperable messaging system which enables blockchains to securely interact with one another. Timber is powered by a network of relayer nodes and is secured by the Byzantine fault tolerant consensus algorithm Tendermint. Anyone is free to join, participate, and use Timber Network. More Info.
